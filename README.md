# MuchStuffPack (MSP)

An **ARK: Survival Ascended** mod that adds much new stuff — custom dinos,
structures, decorations, consumables, resources, craftables, ammunition,
buffs, and seasonal content (including the *Christmas Update 2024*) and more.

> Created by **Sloyserg Mods** — *MSP from Sloyserg Mods*

---

## ⬇️ Download

The complete mod content (~10 GB of Unreal Engine assets) is too large to host

- **CurseForge:** https://legacy.curseforge.com/ark-survival-ascended/mods/muchstuffpack

## 📦 What's inside

- Custom dinos (e.g. Daeodon, Vulture variants)
- Custom structures, building pieces and decorations
- Consumables, resources and craftables
- Ammunition (Fire / Sleep / Good Night arrows, …), buffs and potions
- Seasonal content: Christmas Update 2024, Halloween props, and more
